## 一、积分系统升级

### 新充值套餐
采用更简单的计费方式：**1元 = 1积分 + 额外赠送**

| 充值金额 | 获得积分 | 赠送积分 | 性价比 |
|---------|---------|---------|-------|
| 50元    | 55积分   | 5积分    | 1.10  |
| 100元   | 115积分  | 15积分   | 1.15  |
| 300元   | 370积分  | 70积分   | 1.23  |
| 500元   | 650积分  | 150积分  | 1.30  |

**充值步骤：**
1. 进入个人中心
2. 点击"充值积分"按钮
3. 选择充值档位
4. 扫码支付
5. 上传付款截图
6. 等待管理员审核

### 积分返还机制

**新功能说明：**
当您下架自己发布的交易信息时，系统会自动返还积分。

**返还规则：**
- 返还积分 = 该信息被查看次数 × 1积分
- 未被查看过的信息下架时无返还
- 积分立即到账，可在积分明细中查看

**使用示例：**
```
发布交易信息 → 被查看10次 → 下架 → 自动返还10积分
```

**操作方法：**
1. 进入个人中心 → 我的发布
2. 找到要下架的信息
3. 点击"下架"按钮
4. 系统自动计算并返还积分
5. 提示信息显示返还积分数量

## 二、快捷交易功能

### 绿色"交易"按钮

**位置：**
每个交易信息卡片右下角的绿色按钮

**功能：**
一键查看并复制交易方式，无需进入详情页

**使用步骤：**
1. 在首页浏览交易信息
2. 看到感兴趣的信息，点击"交易"按钮
3. 系统自动：
   - 扣除1积分
   - 复制交易方式到剪贴板
   - 显示成功提示
4. 直接粘贴联系对方

**特殊说明：**
- 如果您已经查看过该信息，再次点击免费
- 积分不足时按钮变灰，无法点击
- 点击后实时更新您的积分余额

**效率提升：**
- 原流程：4步操作
- 新流程：1步操作
- 节省时间：66%

## 七、常见问题

### Q1：为什么下架后有积分返还？
**A：**保护用户权益。您发布的信息被查看时扣除的积分，下架后会返还给您。

### Q2：快捷交易按钮是否每次都扣积分？
**A：**首次查看扣1积分，之后再次点击免费。系统会自动识别您是否查看过。

### Q3：下拉刷新没有反应？
**A：**请确保：
- 页面滚动到最顶部
- 向下拉动距离足够（超过60像素）
- 上一次刷新已完成

### Q4：新充值套餐什么时候生效？
**A：**待Edge Functions部署完成后即可使用新套餐。

### Q5：如何查看积分返还记录？
**A：**个人中心 → 积分明细 → 查找"下架返还"类型的记录
