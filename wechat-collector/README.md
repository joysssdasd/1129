# 微信群消息自动采集工具

自动从微信群采集交易信息，通过AI解析后自动发布到交易平台。

## 功能特点

- 🔄 定时自动采集微信群消息（支持30分钟/1小时/2小时间隔）
- 🤖 AI智能解析交易信息（白银、演唱会门票等）
- 📤 自动调用平台API发布信息
- 📝 支持消息去重，避免重复发布
- 🔒 本地运行，数据安全

## 技术方案

采用**微信数据库读取**方案，定期读取微信PC端本地SQLite数据库。

### 优点
- 不需要Hook微信进程，相对安全
- 数据完整可靠
- 支持历史消息回溯

### 注意事项
- 需要在Windows上运行微信PC客户端
- 首次使用需要获取数据库解密密钥
- 微信运行时需要关闭才能读取数据库（或使用复制方式）

## 安装

```bash
cd wechat-collector
pip install -r requirements.txt
```

## 配置

1. 复制配置文件模板：
```bash
copy config.example.json config.json
```

2. 编辑 `config.json`，填入：
   - 微信数据路径
   - 要监控的群名称
   - 平台API地址和用户ID
   - 采集间隔时间

## 使用

```bash
python main.py
```

## 风险提示

⚠️ 本工具仅供学习研究使用，请遵守相关法律法规和微信使用协议。
